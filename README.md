# 🚀 MarketMind Trading System

MarketMind ek smart trading project hai jo:
- 📊 Stock price data deta hai
- 💰 Buy/Sell simulation karta hai
- 🤖 Prediction feature deta hai

---

## ⚙️ Tech Used
- Python (Flask)
- REST API
- JavaScript

---

## 📡 API

### Home
http://127.0.0.1:10000/

### Price
http://127.0.0.1:10000/price/RELIANCE

---

## 🛠️ Run Kaise Kare

1. Install:
pip install flask flask-cors

2. Run:
python app.py

3. Open:
http://127.0.0.1:10000/

---

## 🔥 Note
Agar broker connect nahi hai to system demo mode me chalega.
